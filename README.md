# PRODIGY_WD_02

## Task 2: Stopwatch Web Application

This project is a simple **Stopwatch Web Application** developed as part of the **Web Development Internship at Prodigy InfoTech**.

The application allows users to start, pause, and reset the stopwatch to accurately measure time intervals.

---

## Features
- Start the stopwatch
- Pause the stopwatch
- Reset the stopwatch
- Real-time time display

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Use the buttons to control the stopwatch
